# Jautājums Modulis / Ekrānforma Datums
1 Kā tiek reģistrēta iesnieguma izskatīšanas "atlikšana"?
Statuss "Atlikts" tiek uzstādīts no šķirkļa "Atlikt / Atteikt iesniegumu" → "Atlikt apstrādi" → poga "Saglabāt". Nepieciešams precizēt: vai jāreģistrē dokuments par termiņa pagarinājumu, vai pietiek ar statusa maiņu? 		Atlikt / Atteikt iesniegumu 26.05.2026
2 Kā reģistrēt pārtraukuma periodu?
26.05.2026 — jauna ekrānforma "Izsniegtie A1 sertifikāti". Pārtraukuma periods šobrīd tiek pievienots šķirklī "Sagatavotās atbildes" → sadaļa "Pārtraukuma datums". Jāprecizē: vai pārtraukuma periods ir saistāms tieši ar konkrētu izsniegtu A1 sertifikātu? 		A1 sertifikāti / Sagatavotās atbildes 26.05.2026
3 Kā reģistrēt saņemto A1 sertifikātu no citas dalībvalsts?
Šķirklis "A1 sertifikāti" → bloks "Saņemtie A1 sertifikāti" (tabula ar Saņemšanas datums, Periods no, Periods līdz). Jāprecizē ievades mehānisms un vai nepieciešams piesaistīt oriģināldokumentu. 		A1 sertifikāti 26.05.2026
